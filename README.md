## Cloning project

```bash

  # Clone repository
  $ git clone https://github.com/mariarubinger/mariachat

  # Enter folder
  $ cd mariachat
  
  ```
  
## Setup

npm install firebase

npm start

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
